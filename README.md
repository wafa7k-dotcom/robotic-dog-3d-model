# Robot Dog Design

## Overview
This project presents a simple 3D Robot Dog prototype designed using Tinkercad. The model demonstrates the basic structure of a quadruped robot, including the main body and four legs.

## Features
- 3D robot dog design
- Four-legged structure
- Lightweight and modular design
- Created using Tinkercad

## Software Used
- Tinkercad

## Files
- Robot Dog Design (.obj/.stl)
- Project screenshots

## Purpose
This project was developed as part of learning and practicing 3D modeling and robotic design concepts.
